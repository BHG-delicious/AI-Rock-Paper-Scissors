# 🤖 AI 剪刀石頭布 · 競賽版

本專案為 AI 對戰平台，完整實作「五階段競賽規則」，讓兩位 AI 透過人類中介進行高規格猜拳博弈。

## 🎯 核心特色

- **五階段完整流程**：規則宣讀 → 比賽準備 → 比賽中途 → 比賽結果 → 交換復盤
- **靈活的策略選擇**：每位選手可自由選擇「AI 自選策略」或「人類指定策略」
- **智慧提示詞生成**：每回合自動產生詳細提示詞，包含歷史記錄、對手模式分析、上一回合 JSON
- **心理戰支援**：允許 AI 在 `reason` 中釋放假訊息，實現策略性欺騙
- **復盤與交換**：賽後自動生成復盤提示詞，並支援雙方復盤內容交換
- **現代化 UI**：參考「大家政便當廠」風格，卡片式設計、直覺步驟引導

## 🚀 如何使用

### 1. 規則宣讀
- 頁面會顯示完整規則文字，點擊「複製規則」按鈕，將規則分別貼給兩位 AI。
- 請 AI 回覆「我已完整閱讀比賽規則並同意參賽」。
- 勾選確認方塊後，進入下一階段。

### 2. 比賽準備
- 設定兩位 AI 選手（名稱、模型）。
- 每位選手可獨立選擇策略產生方式：
  - **AI 選擇策略**：系統會產生提示詞，複製給 AI 後，AI 會回覆策略名稱，貼入輸入框。
  - **指定策略**：手動輸入策略名稱，並可點擊「生成提示詞」產生對應的指定策略提示詞。
- 雙方都填妥策略名稱後，「開始比賽」按鈕啟用。

### 3. 比賽中途
- 系統自動產生雙方提示詞（含歷史記錄、對手分析、上一回合 JSON）。
- 將提示詞分別複製給對應的 AI，並將 AI 回覆的 JSON 貼回輸入框。
- 點擊「解析回應」，系統自動判定勝負並更新比分。
- 繼續下一回合，直到達成勝利條件。

### 4. 比賽結果與復盤
- 系統宣布獲勝者，並自動產生復盤提示詞（含最後一回合對手 JSON）。
- 將復盤提示詞複製給雙方 AI，貼上它們的復盤內容並儲存。

### 5. 交換復盤結果
- 系統會將雙方復盤內容互相提供給對方（產生交換提示詞）。
- 複製提示詞給對應 AI，讓它們閱讀對方的復盤，完成最終交流。

## 🧠 策略與心理戰

- 提示詞內建「對手出招分析」與「策略建議」，鼓勵 AI 進行反預測。
- `reason` 欄位可自由發揮，可故意誤導對手。
- 每回合完整揭露雙方上一回合的 JSON，便於動態調整。

## 🛠 技術實現

- **前端**：純 HTML5 + CSS3 + JavaScript（ES6）
- **無後端**：所有邏輯於瀏覽器端執行，資料儲存於 `localStorage`
- **依賴**：Font Awesome 圖示庫（CDN）
- **相容性**：現代瀏覽器（Chrome, Firefox, Safari, Edge）
